# Moses Nyanzi | Strategy, Automation, Growth

Personal consulting website for Moses Nyanzi, an INSEAD MBA and Actuary specialising in capital strategy, M&A due diligence, and operational transformation.

## Live Site

Strategy. Rigour. Results.

## Tech Stack

- **Framework:** React 18 + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Colour Scheme:** Emerald green + gold accents

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── components/         # React components
│   ├── HeroSection     # Main hero with name and tagline
│   ├── ValueProposition# Problems I Solve section
│   ├── AboutSection    # Your Advisor section
│   ├── ExpertiseSection# What I Bring (4 pillars)
│   ├── ProofSection    # Results I've Delivered
│   ├── EngagementSection# Contact CTA and footer
│   ├── DiagnosticFlow  # Interactive contact modal
│   └── ui/             # shadcn/ui components
├── pages/
│   └── Index.tsx       # Main page layout
└── index.css           # Global styles and design system

Skills/
└── MASTER_CV.txt       # Comprehensive CV / Skills Bank
```

## Key Sections

1. **Hero** - Name, tagline, problem recognition
2. **Problems I Solve** - Three value propositions
3. **Your Advisor** - What you get, how I deliver
4. **Expertise** - 4 pillars (Strategy, Operations, Risk, Analytics)
5. **Proof** - Quantified achievements
6. **Engage** - Contact CTA with diagnostic flow
